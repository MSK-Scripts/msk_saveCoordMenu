## Description
* Teleport to the saved coords in this menu
* Set Groups who can open this menu
* Set default coords in config.lua
* Every Player can save own coords for themself [saved in database]
* Discord Logs

## Requirements
* ESX 1.1, ESX 1.2, ESX Legacy
* NativeUI
* oxmysql
* msk_core
